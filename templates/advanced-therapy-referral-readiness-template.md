# Advanced Therapy Referral Readiness Template

Status: private worksheet template; do not commit a filled patient-identifying
copy
Purpose: organize records before asking a hematologist or referral center about
HSCT, gene therapy, CRISPR therapy, luspatercept, mitapivat, hydroxyurea, or
trial screening

## Privacy Boundary

Do not paste raw reports, names, addresses, phone numbers, exact birth dates,
hospital IDs, accession numbers, signatures, photos, QR codes, barcodes, or
local file paths into a public copy.

Use this privately first. If a public summary is needed, extract only minimum
de-identified facts and run the public case-data release checklist.
