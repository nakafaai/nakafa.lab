# Research Data Snapshots

This folder stores small reproducible snapshots generated by repo scripts.

## Rules

- Do not store identifiable patient data here.
- Keep generated files small and source-linked.
- Prefer JSON outputs from `scripts/fetch_pubmed.py` and
  `scripts/fetch_clinical_trials.py`.
- Prefer official chemistry APIs for compound identity snapshots.
- Do not commit raw browser HTML. Keep source URLs, structured API metadata,
  and concise `.txt`/table extracts instead.
- Use dated filenames so future snapshots can be compared.

## Current Snapshot Types

- `literature/pubmed/` - PubMed search snapshots.
- `literature/fulltext/ncbi_books/` - NCBI Bookshelf text extracts.
- `literature/fulltext/plos/` - PLOS text extracts.
- `literature/fulltext/pmc/` - PMC text extracts.
- `literature/fulltext/pubmed/` - PubMed page text extracts when page-level
  context matters.
- `literature/supplementary/plos/` - PLOS supporting-data snapshots used for
  raw assay-table extraction.
- `registries/clinicaltrials/` - ClinicalTrials.gov API snapshots.
- `chemistry/pubchem/` - PubChem compound property snapshots.
- `chemistry/chembl/` - ChEMBL molecule search snapshots.
- `chemistry/pubchem/bee-constituents/` - PubChem snapshots for the
  bee-derived seed panel.
- `chemistry/chembl/bee-constituents/` - ChEMBL snapshots for the
  bee-derived seed panel.
- `chemistry/pubchem/curcuminoid-analogs/` - PubChem snapshots for natural
  curcuminoids used as analog comparators.
- `chemistry/chembl/curcuminoid-analogs/` - ChEMBL search snapshots for the
  curcuminoid analog lane.
- `chemistry/pubchem/natural-product-hbf/` - PubChem identity snapshots for
  natural-product HbF leads.
- `chemistry/chembl/natural-product-hbf/` - ChEMBL identity and activity
  snapshots for natural-product HbF leads.
- `chemistry/pubchem/quercetin-analogs/` - PubChem identity snapshots for the
  quercetin analog HbF lane.
- `chemistry/chembl/quercetin-analogs/` - ChEMBL search, molecule, and activity
  snapshots for the quercetin analog HbF lane.
- `chemistry/pubchem/hudep2-hbf-hits/` - PubChem snapshots for HUDEP2
  small-molecule HbF benchmark hits.
- `chemistry/chembl/hudep2-hbf-hits/` - ChEMBL snapshots for HUDEP2
  small-molecule HbF benchmark hits.
- `literature/pubmed/` and `registries/clinicaltrials/` - hepcidin,
  ferroportin, `TMPRSS6`, erythroferrone, vamifeport, minihepcidin, SLN124,
  REGN7999, and related iron-axis snapshots.
- `chemistry/pubchem/iron-axis/` and `chemistry/chembl/iron-axis/` - compound
  identity snapshots for iron-axis comparators such as vamifeport and
  rusfertide.
- `regulatory/bpom/` - BPOM Cek Produk snapshots for Indonesian access checks.
- `regulatory/efsa/` - EFSA-linked safety extracts used for safety boundaries.
