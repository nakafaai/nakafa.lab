## Current Findings

The May 12 snapshot returned six active or active-not-recruiting mitapivat query
records. The record set includes adult NTDT, adult TDT, pediatric TDT, and
pediatric NTDT thalassemia contexts, but also one sickle-cell record that must
be treated as query-adjacent noise.

The jurisdiction map currently supports these safe statements:

- United States: FDA adult thalassemia approval exists for Aqvesme.
- Saudi Arabia: SFDA adult thalassemia registration exists for Pyrukynd.
- European Union: EMA positive CHMP variation evidence exists, but the current
  European Commission authorisation state must be verified before access
  claims.
- Indonesia: local BPOM registration was not verified in this public pass;
  this is not proof of non-registration.
